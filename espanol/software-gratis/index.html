Software comercial para descargar gratuitamente

Este tipo de software es necesario para acceder a ciertos documentos del Gobierno que están en formato de documento portátil (PDF, sigla en inglés) o para usar archivos de audio y video, entre otros. Si usted hace clic en uno de esos enlaces, pasará a un sitio comercial que no pertenece al Gobierno.

Formato de documento portátil (PDF)

Algunos documentos de los sitios web del gobierno están en formato de documento portátil. El PDF le permite ver las publicaciones en su formato original, independientemente de qué tipo de computadora, navegador o sistema operativo utilice para acceder al Internet. Para ver, navegar e imprimir un archivo en PDF, deberá descargar el programa gratuito Acrobat Reader. Las instrucciones y la guía de solución de problemas están disponibles en el sitio web de Adobe (en inglés). Si utiliza un lector de pantalla, puede descargar el software de Adobe para crear archivos PDF más accesibles (en inglés).

Archivos de Video y Audio

Estos archivos, cuya extensión puede ser (".ram", ".ra", ".rm", ".rmm", ".rp", ".rms", ".rt"), ".rmj"), requieren que usted instale y use el software RealPlayer. Este software es gratis y se puede descargar desde RealNetworks (en inglés). Este software le permite transmitir (stream) el archivo de audio o video, lo que significa que usted comienza a escucharlo o verlo sin tener que esperar que se descargue completamente. Para escuchar archivos de audio, su computadora deberá tener una tarjeta de sonido, parlantes o auriculares.

Programas lectores de Distribución Realmente Sencilla (RSS, sigla en inglés)

Las notificaciones (feeds) de RSS, cuya extensión puede ser (".xml", ".rss", ".sfm", ".cfm", ".rdf", ".aspx", o ".php"), requieren que usted instale y use un lector de RSS. Un programa lector de la RSS le permite suscribirse a un canal de RSS. Hay muchos programas disponibles; algunos son gratuitos y algunos están disponibles para la venta. Los canales de RSS son comúnmente utilizados en los blogs, sitios web de noticias y otros sitios que ofrecen contenido que se actualiza con frecuencia. Un programa lector de RSS reúne el material de los sitios web que usted ha indicado y le hace llegar sus actualizaciones. Es un formato conveniente porque le permite ver los nuevos contenidos de múltiples canales en un solo lugar.

Podcasts

Estos archivos, cuya extensión puede ser (".xml" y ".php"), requieren que usted instale y use un software de podcatching (un tipo de lector). Este software está disponible para ser descargado de forma gratuita desde Juice (en inglés) para usuarios de PC, Mac y Linux y iTunes para PC y Mac. Existen otras opciones de software de podcatching. Podcasting es una forma de transmitir (stream) audio a través del Internet al suscribirse a una fuente de estos archivos, por lo general de formato MP3. Usted puede suscribirse a las fuentes utilizando software de podcatching, que periódicamente busca y descarga el nuevo contenido de forma automática. Cualquier reproductor de audio digital o computadora que tenga software de reproducción de audio puede transmitir podcasts.

Otros archivos de audio

Algunos archivos de audio de los sitios web del Gobierno tienen una extensión (".au"). Estos archivos de audio se pueden transmitir (stream) con un reproductor de sonido estándar incluido en el sistema operativo de su computadora, por ejemplo, Windows Media Player (en inglés) para Windows, QuickTime (en inglés) para Macintosh y XPlayGizmo (en inglés) para Unix. Para escuchar archivos de audio, su computadora deberá tener una tarjeta de sonido, parlantes o auriculares.
